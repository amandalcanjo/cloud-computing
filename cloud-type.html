Cloud Types

The cloud is divided into three categories that need to be understood to assess information security and that may be required by the customer.

Public Cloud

The public cloud is defined when resources are shared by a third party through a computing service. The concern in this case is with data confidentiality. This type of cloud has been used frequently for business applications and dissemination of marketing material. All infrastructure is owned and managed by the cloud provider.

Private Cloud

It can be physically structured in a data center or available from a third-party provider. Defined when the use is exclusive to the contractor through server virtualization. It has a higher cost, but it is used to store strategic data, has faster responses with low latency and high security.

Hybrid Cloud

Set when the customer uses both types of storage. It has greater flexibility, allowing data transfer between platform types and being able to allocate extra resources from a supplier on demand.

Multicloud

consists of using two or more cloud computing frameworks. The combination aims to use different providers of cloud services to increase computing capacity, security and availability or to optimize expenses. The use of different clouds also allows companies to keep up-to-date and use the latest technological innovations developed by several cloud providers, not just one.
